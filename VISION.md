# THE YVN CODE — Vision

## Что это
THE YVN CODE — это редактор кода под моим брендом,
основанный на открытом исходном коде VS Code (через VSCodium).


## Чем будет отличаться от обычного VS Code
- Свой бренд и логотип
- Без телеметрии Microsoft
- С моими кастомными разширениями итд

## Планируемые фичи
- [x] Свой бренд (имя в скриптах сборки, product.json, метаданные)
- [x] Свой логотип и иконки
- [ ] Встроенный AI (Codex / Claude Code / Anima Yvn)
- [ ] Предустановленные расширения
- [ ] Визуальные настройки для своих разных сервисов

## Технический стек
- Основа: VSCodium скрипты сборки (MIT лицензия)
- Сборка: WSL2 (Linux), VPS (CI/CD)
- Маркетплейс расширений: Open VSX Registry

### Этап 1 — Брендинг (сейчас)

- Встроенный Codex/Claude
